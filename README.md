# travel-agent

根据https://github.com/datawhalechina/hello-agents完成的 Python 旅行agent。

## 项目文件

- `agent_char1.py`：主程序入口
- `requirements.txt`：Python 依赖列表

## 环境准备

先安装依赖：

```bash
pip install -r requirements.txt
```

运行前需要配置以下环境变量：

- `API_KEY`
- `BASE_URL`
- `MODEL_ID`
- `TAVILY_API_KEY`

## 运行方式

```bash
python agent_char1.py
```
