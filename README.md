# travel-agent

这是一个基于 Python 的旅行助手项目。

## 项目文件

- `agent_char1.py`：主程序入口
- `requirements.txt`：Python 依赖列表

## 环境准备

先安装依赖：

```bash
pip install -r requirements.txt
```

运行前需要配置以下环境变量：

- `MINIMAX_API_KEY`
- `MINIMAX_BASE_URL`（可选）
- `MINIMAX_MODEL_ID`（可选）
- `TAVILY_API_KEY`

## 运行方式

```bash
python agent_char1.py
```
